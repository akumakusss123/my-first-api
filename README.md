# Мое Первое API

## Описание
Это мой первый проект API на ASP.NET Core. Я научился создавать веб-сервисы!

## Что умеет этот проект:
- Хранить список имен
- Добавлять новые имена
- Удалять имена
- Искать имена

## Как запустить:
1. Скачай Visual Studio
2. Открой папку src
3. Нажми F5
